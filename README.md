# Wrist Roulette PWA

This is a local-only iPhone/desktop web app.

## What it does
- People share the same app link.
- Each person saves their own watches only on their own phone/browser.
- No shared Render database.
- No login.
- Photos are compressed before saving.
- Export/Import backup lets users move or protect their watch collection.

## Deploy on Render
1. Create a new GitHub repo.
2. Upload these files.
3. In Render, create a new Static Site.
4. Connect the GitHub repo.
5. Build command: leave blank
6. Publish directory: `.`
7. Deploy.

## iPhone install
1. Open the Render app link in Safari.
2. Tap Share.
3. Tap Add to Home Screen.
4. Tap Add.

Each iPhone keeps its own private watch list.

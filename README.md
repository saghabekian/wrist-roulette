# Wrist Roulette PWA v2

## New in v2
- Select individual watches with checkboxes.
- Delete selected watches only.
- Red X still removes one watch at a time.
- Fancy luxury-watch app icon.
- Local-only storage per phone/browser.
- Export/Import backup.

## Deploy on Render
Upload the contents of this folder to your GitHub repo, replacing the old files.
Render should redeploy automatically.

If the old app stays cached on your phone:
1. Open the app.
2. Force close it.
3. Reopen after a minute.
4. If needed, delete the Home Screen icon and add it again from Safari.
